GitHub 任意の連携

チケット駆動ワークフロー（ticket_driven_workflow）を使う場合のみ。プロジェクト単位の設定です。

この連携を使う
GitHub はコア設定ではありません。未接続でも、翻訳などの単発コマンドやスケジュール実行は利用できます。なお各メンバーの GitHub 認証（App / トークン）はメンバー編集画面で設定します。
